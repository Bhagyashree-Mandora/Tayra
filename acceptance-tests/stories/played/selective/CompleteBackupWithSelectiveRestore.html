Tayra performs selective restore

Narrative

Adam, MongoDB admin, performs backup of a replica set on an ongoing basis.
On a not so great day, Dave, the developer, notices that the profile collection in the users database had inconsistent data due to application issues.
So, Dave approaches Adam and requests him to restore all other databases except the profile collection in user database.
